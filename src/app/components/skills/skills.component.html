Web Technology, Frameworks and Languages I Used for web development

Node JS

Node is a serverside development environment that I used for Back-end development. It also uses JavaScript on the server


Angular 4/5

Is a client side development framework. It uses TypeScript and with my background with C#.net I was able to understand and used this accordingly


MongoDB/Mongoose

MongoDB is a No-SQL Database while I used Mongoose for model abstraction, data manipulation and validation


Express

Is a web application framework I used for creating my Web API routes, handling response and requests through RESTful Web Services and for setting up the web server that will run in Node


JQuery

Is a JavaScript library I used for DOM manipulation. Sometimes I used Embedded Javascript (EJS) as my templating engine combined with JQuery


Bootstrap 3/4

Is a frontend framework I used to create responsive and amazing design for my web site and web applications


Amazon Web Services

I used Amazon Web Services for deploying a web application in the web. I mostly use AWS Elastic Compute Cloud (EC2) to deploy my MEAN app including Classic Elastic Load Balancer, Route 53, Elastic IP address and AWS Certificate Manager for managing SSL Certificates


Heroku

I also use Heroku for deploying my website and web application in the web. It's easier to use than Amazon Web Services since it's a Platform as a Service, PaaS
